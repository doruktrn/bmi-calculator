# BMI Calculator

A Python program that calculates Body Mass Index (BMI) for multiple users and stores BMI history using lists.

## Features
- Calculates BMI using weight and height
- Categorizes BMI results
- Stores BMI records in a list
- Supports multiple users
- Uses functions, loops, and selection statements

## Technologies
- Python

## Example Output

BMI for Josh: 27.55 -> Category: Overweight
